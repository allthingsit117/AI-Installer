# 🧠 AI-Installer

**AI-Installer** is a simple, beginner-friendly Python script that automates the installation of popular tools required to run a **local AI model** on a Linux server.

This script installs:

- 🐳 **Docker**
- 🌐 **Open WebUI**
- 🤖 **Ollama** (for running local LLMs like LLaMA, Mistral, etc.)

---

## 🔧 Features

- ✅ Installs all dependencies automatically  
- 🐍 Written in pure Python — no extra modules required  
- 🧠 Run local AI models like `llama2`, `mistral`, or others with Ollama  
- 📡 Launches the Open WebUI container at [`http://localhost:8080`](http://localhost:8080)

---

## 🚀 How to Run

```bash
sudo python3 AI-Installer.py
```

Once complete:

- Access the Web UI in your browser at **http://localhost:8080**
- Pull and run an AI model using:

```bash
ollama pull <model_name>
ollama run <model_name>
```

Example:

```bash
ollama pull llama2
ollama run llama2
```
